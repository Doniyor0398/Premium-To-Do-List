> Управляйте своими делами красиво и эффективно.

---

## ✨ Особенности

- ✅ Добавление, завершение и удаление задач
- 🗂️ Категоризация по приоритетам (высокий / средний / низкий)
- 🔍 Фильтрация и поиск по задачам
- 💾 Сохранение данных в `localStorage` (не пропадают после перезагрузки)
- 📱 Адаптивный дизайн (работает на мобильных устройствах)
- 🌙 Тёмная тема (переключается в один клик)

---

## 🛠️ Технологии

- **HTML5** – семантическая структура
- **CSS3** – Flexbox / Grid, анимации, переменные
- **Vanilla JavaScript** – без фреймворков (чистый DOM, события, хранилище)
